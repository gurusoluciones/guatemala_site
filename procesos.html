Proceso Web One Page

COMERCIAL:
- Asesor debe llenar el Brief Mínimo Obligatorio
- Si asesor no proporciona información para la página web esta se va a No Conformidad, tener presente que este tipo de páginas no tiene relevamiento. 

PRODUCCIÓN:
- Gestión de Clientes accede al BMO y elabora la hoja de redacción 
- Envía a diseñador pedido para elaborar la Página Web
- Una vez elaborado el diseño pasa a Etapa QA en Sales Force
- Gestión de Clientes revisa página, compra el dominio y publica la web
- Gestión de clientes configura buzones de correos
- Tiempo aproximado de producción 10 a 15 días.

Plazos de Relevamiento

GESTIÓN DE CLIENTES
- A excepción de la Web EMPRENDE todas las Páginas Web pasan por la etapa de relevamiento
- EL cliente tiene un plazo de 7 días utiles
- En el plazo de 7 días se le deben efectuar un mínimo de 3 llamadas al cliente
- Si Cliente no proporciona más información se elabora hoja de redacción con información del BMO.
- Si Cliente solicita más plazo para entregar información, se envía a etapa Aplazado por el Cliente un máximo de 10 días, sino cumple se procede como paso anterior.
- Si Cliente durante relevamiento manifiesta que no desea la Web se envía a No Conformidad.

Cliente no aprueba Publicación

GESTIÓN DE CLIENTES 
- Si Cliente no aprueba publicación se le da un plazo máximo de 10 días calendario
- Si Cliente insiste en no publicar página web se envía a No Conformidad para que asesor apoye a conseguir la aprobación. 

Cliente en Aplazado no completa información

GESTIÓN DE CLIENTES
- Si Cliente no proporciona más información se elabora hoja de redacción con información del BMO si esta no fuera suficiente se elabora hoja de redacción con información de otros productos contratados o cartera.
- Si la información del BMO es insuficiente y no cuenta con otros productos se envía a No Conformidad.

Procesos productivos

Cliente no da Acceso a GMB

COMERCIAL
- Asesor debe proporcionar los datos de acceso al GMB del cliente.

GESTIÓN DE CLIENTES
- Si se tienen los datos de acceso al GMB se realiza la sincronización.
- Si no se tienen los datos de acceso, se llama y/o escribe al cliente y se solicita.
- Si Cliente no cede la administración se solicita a Google que ceda la administración.

Cliente no da acceso a su Fan Page

COMERCIAL
- Si el Cliente no tiene Fan Page, Gestor de Cliente le crea nueva FP.
- Si el Cliente ya tiene Fan Page el Asesor debe proporcionar los datos de acceso al Fan Page en el Brief. 

GESTIÓN DE CLIENTES
- Si se tiene el acceso a la Fan Page se realiza sincronización.
- Si no se tienen los datos de acceso se solicita al cliente que los proporcione indicándole la importancia de entregarlos para realizar la sincronización de su Fan Page.
- Si cumplidos los 5 días cliente no cede la Administración de su Fan Page se entrega +Geo sin sincronizar.
- Cuando el cliente sin vinculación llega a la gestión del Client Manager es él quien solicita la administración, si Cliente la cede, se sincronica, si no la cede queda el +Geo sin sincronizar Fan Page.
* En ningún caso se crea una nueva Fan Page.

No sincroniza dirección

GESTIÓN DE CLIENTES
- Si no se sincroniza dirección se envía correo a Google para forzar sincronización.

Bajo % de Optimización

GESTIÓN DE CLIENTES
- Se considera bajo cuando el % es inferior al 91%, el objetivo es llegar al 100%.
- Se procederá a optimizar el producto considerando el informe semanal de estatus de producción.
En el se identifcan las fichas con el menor % de completitud y nivel de sincronización.
- Se priorizarán las fichas que estén próximas a renovarse, con el fin de que los asesores visiten a los clientes y aminoren la probabilidad de reclamos.

GESTIÓN DE CLIENTES
- Se deben tomar los logos y fotografías de la figuración de PACOM.
- También se pueden buscar las fotografías de las redes sociales del cliente o página web.

Envío de información para Figuración

CLIENTE CARTERA:
- Asesor debe llenar el contenido textual de las figuraciones en Sales Force, nunca en Brief.

CLIENTE POTENCIAL:
- Asesor debe llenar el contenido textual de las figuraciones en Sales Force. 
- Si asesor no ingresó textos en Sales Force, se envía a No Conformidad los movimientos afectados.

Envío de información gráfica para Figuración

- Se subirán a Salesforce de la siguiente manera:
   Cuenta – Oportunidad – Cloud Documents
- Nombrar de esta manera:
LOGO. jpg (gif,pdf, tif) , si cliente no tiene Logo, indicar en el Brief Crear Logo Tipográfico o Genérico.
GALERIA.zip . El Zip debe de contener las CINCO (5) imágenes. Si tienen mas secciones identificar el nombre de la sección. Ejm. GALERIA-dentistas.ZIP

Google Ads

Procesos productivos

Asesor no envía Brief

- Si el asesor comercial no proporciona el brief no se puede elaborar la campaña
Se debe generar la No Conformidad al asesor comercial y devolver el movimiento.
- El Brief debe contener los datos de contacto del cliente y datos puntuales de campaña vendida.

Reportes de Campañas

- Todas las campañas cuentan con un reporte elaborado en Google Data Studio, los aseseores comerciales pueden acceder a la URL de reporte que se encuentra en el MCC en la etiqueta de la campaña.
- Del mismo modo pueden consultar los resultados de las campañas en Portal del Cliente consultando por cuenta de cada cliente.

Elaboración de Semáforo de Campañas

- El equipo de S&P es el responsable de elaborar el semaforo de estatus de campañas cada semana.
El Account Manager trabaja las optimizaciones de las campañas en base a los resultados del semáforo.

Maquetación de Campañas nuevas o Renovaciones

Siempre que se Maquete (configure) una nueva campaña o renueve en el MCC se deben considerar los siguientes pasos:

1.- Configuración de campaña en el MCC (Maquetación).

2.- Creación de Reporte en Google Data Estudio, usando la plantilla predefinida, la plantilla debe contener los datos de contacto del Account Manager. Una vez generado el URl del reporte se debe incluir en una etiqueta en el MCC para que esté a disposición de quien necesite el reporte.

3.- Cambio de etapas del Movimiento en Sales Force hasta llevarla a Entregado.

4.- Para el caso de las renovaciones se debe de validar que la campaña vigente tenga correctamente configurados los pasos antes mencionados y validar su situación en el semáforo.

Gestión de la Producción de campañas de Google Ads                           

En Google Ads tenemos 2 tipos de Campañas:

a) Gama Alta (comprende aquellas campañas que invierten desde 380 Dólares mensuales a más) y
b) Gama Baja (comprende aquellas campañas que invierten menos de 380 dólares mensuales.

- Para las campañas de Gama alta se contempla que el Account Manager llame al cliente para relevar información acerca de la campaña que quiere lanzar.
El asesor comercial debe proporcionar los datos de contacto del cliente en el Brief (Nombre, cargo, teléfono celular, horario de contacto, etc.) y algunos datos acerca de la campaña vendida. 

- Para las Campañas de Gama Baja el Asesor comercial debe proporcionar en el Brief la información referente a la campaña que su cliente quiere lanzar.

Si asesor no brinda información sufiiente en el Brief se envía a No Conformidad. (No hay contacto telefónico para gama baja).

Acciones de Control, Seguimiento y Optimización

El Account Manager cuenta con el archivo de control donde se especifica la situación de cada campaña, métricas importantes y acciones de mejora recomendados, en base a este archivo debe de actuar optimizando todas aquellas que lo requieran.
